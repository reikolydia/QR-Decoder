# QR-Decoder

Python terminal based QR code decoder, or a URL-encoded string decoder. Works well with characters outside ascii, like CN chars. code39 barcodes may work occasionally

Uses these libraries:
pillow
rich
pyperclip
pyrxing


Usage:
Have a image copied to clipboard, on windows the print screen capture works great, I don't have other systems to test at the moment, so this code is largely for windows.
Then run script, then Y

If you already ran the script, you can either CTRL-C or press ENTER to input an empty string then N finally Y
